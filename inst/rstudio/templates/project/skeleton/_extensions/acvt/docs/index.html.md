Welcome to the documentation for the `Academic-CV Template`. This is a comprehensive **Quarto Project Extension** designed to help you create professional academic CVs and cover letters using the Typst typesetting system.

## Project Extension

Unlike simple document templates, this is a full-featured project extension. It integrates:

*   **Project Type:** Activated via `project: type: acvt` to handle the build pipeline.
*   **Custom Format:** `acvt-typst` for high-quality PDF generation.
*   **Filters & Shortcodes:** Powerful tools to fetch data, process content, and automate layout.
*   **Typst Templates:** Modular, customizable styling.

## Why Use This Extension?

*   **Data-Driven:** Keep your CV content in Google Sheets and let the extension handle the formatting.
*   **Automated Layout:** Use shortcodes like `{{< cv-section >}}` to map your data to professional 2-column layouts.
*   **Flexible Styling:** Customize fonts, colors, and content using YAML and Typst inline code.
*   **Publication Management:** Generate categorized publication lists automatically from BibTeX, BibLatex, CSL-JSON, RIS, YAML and XML.
